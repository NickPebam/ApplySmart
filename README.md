# ApplySmart 🚀

ApplySmart is a modular, production-ready application designed to help users analyze resumes and apply smartly using AI-driven insights.

This repository currently contains **Phase 1** of the project.

---

## Phase 1 – Authentication Service ✅

A standalone authentication service built with Spring Boot.

### Features
- JWT-based authentication
- Role-based access control
- Secure password handling
- Swagger / OpenAPI documentation
- Configuration via environment variables

---

## Tech Stack
- Java 17
- Spring Boot 3
- Spring Security
- JWT
- OpenAPI (Swagger)
- MySQL / PostgreSQL
- Maven
